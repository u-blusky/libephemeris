# LibEphemeris v1.0.0-alpha.3 — Horizons API Backend

**Release date:** 2026-03-26

---

## Highlights

This release adds a **NASA JPL Horizons API backend** that enables ephemeris
computation with zero local data files — just `pip install libephemeris` and
start calculating. Combined with the LEB2 compressed format from alpha.2, the
library now supports a three-tier fallback chain:

```
LEB (precomputed, ~5us)  ->  Horizons API (~300ms)  ->  Skyfield/DE440 (~7ms)
```

## Calculation Modes

Four modes, configurable via `set_calc_mode()` or `LIBEPHEMERIS_MODE` env var:

| Mode | Flow | Use case |
|------|------|----------|
| `"auto"` | LEB -> Horizons (if no DE440) -> Skyfield | Default, works everywhere |
| `"horizons"` | Horizons only | Zero-install, serverless |
| `"leb"` | LEB only | Maximum performance |
| `"skyfield"` | Skyfield/DE440 only | Offline, maximum precision |

## Horizons Backend Architecture

**New module:** `libephemeris/horizons_backend.py` (~720 lines)

- `HorizonsClient` — HTTP client with LRU cache (4096 entries), parallel fetch
  (ThreadPoolExecutor, 8 workers), retry with exponential backoff
- Full geocentric apparent pipeline: light-time iteration, gravitational deflection
  (Sun + Jupiter + Saturn, PPN model), stellar aberration, frame rotation
- Analytical dispatch for Mean Node, Mean Apogee, Uranians (no HTTP needed)
- Velocity via numerical differentiation of apparent position

**Supported:** Sun-Pluto, Earth, Chiron, Ceres-Vesta, Mean Node, Mean Apogee, Uranians (helio)
**Fallback to Skyfield:** True Node, Oscu/Interp Apogee, fixed stars, planetary moons, SEFLG_TOPOCTR

## Precision

Measured against Skyfield/DE440 reference:

| Mode | Tests | Max error |
|------|-------|-----------|
| Geocentric (all flags) | 12,500 | **0.0003"** |
| Heliocentric | 2,600 | 0.027" (systematic offset) |
| vs LEB2 | 9,000 | 0.0002" |

**Known limitations:**
- Heliocentric ~0.01-0.03" offset (Horizons Sun center vs Skyfield SSB-Sun)
- Moon velocity ~0.001 deg/day (numerical vs analytical derivative)
- Requires internet connectivity
