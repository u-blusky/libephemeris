# Manuale LibEphemeris (Italiano)

Introduzione ai calcoli astronomici ed astrologici per principianti.
15 capitoli che coprono il cielo, il tempo, le coordinate, le effemeridi,
i pianeti, la Luna, le case astrologiche, le stelle fisse, le eclissi,
lo zodiaco siderale, i corpi minori e le ricette pratiche.

> Questo manuale non richiede conoscenze pregresse di astronomia o programmazione.
